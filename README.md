# 🕵️ Crime Analytics Dashboard Suite

> An interactive Power BI dashboard suite for analyzing crime patterns, victim-perpetrator relationships, and law enforcement efficiency across the United States.


---

## 📖 Project Overview

This project consists of **three interactive Power BI dashboards** developed using a **US Crime Dataset** to analyze crime patterns, victim-perpetrator relationships, and law enforcement efficiency.

The dashboards provide actionable insights through:

- 📊 Data Visualization & KPI Monitoring
- 📈 Trend Analysis
- 🗺️ Geographic Mapping
- 🔍 Demographic Profiling

The solution helps stakeholders understand crime behavior, identify crime hotspots, analyze demographic patterns, and evaluate agency performance for better decision-making.

---

## 📊 Dashboard Modules

### 🗺️ Dashboard 1: US Crime Overview

**Objective:** Provide a high-level overview of crime incidents across different states, cities, and years.

#### Key KPIs
| Metric | Value |
|---|---|
| Total Incident Count | 638K |
| Average YoY Growth | Tracked |
| Top State by Incidents | Identified |

#### Visualizations
- 📅 Incident Count by Year (Trend Analysis)
- 🏙️ Incident Count by City
- ✅ Crime Solved vs Unsolved Analysis
- 🗺️ State-wise Crime Distribution Map
- 🌳 Incident Count by Month (Treemap)

#### Filters Available
`Year` · `State` · `City` · `Crime Solved`

#### Insights Generated
- Identify crime trends over time
- Detect cities with highest crime rates
- Analyze monthly crime distribution
- Compare solved vs unsolved incidents
- Locate crime hotspots geographically

---

### 👥 Dashboard 2: Victim vs Perpetrator Analytics

**Objective:** Analyze demographic characteristics and relationships between victims and perpetrators.

#### Key KPIs
| Metric | Value |
|---|---|
| Average Victim Age | 35.03 Years |
| Average Perpetrator Age | 20.32 Years |
| Victim-to-Perpetrator Ratio | 0.67 |
| Most Affected Gender | Male |

#### Visualizations
- ♂️♀️ Gender Count Analysis
- 📊 Victim Age Distribution Histogram
- 📦 Victim vs Perpetrator Age Box Plot
- 🔗 Relationship Distribution Analysis
- 🌍 Race-wise Victim and Perpetrator Comparison

#### Filters Available
`Race` · `Crime Type` · `Year` · `Victim Gender` · `Relationship`

#### Insights Generated
- Most affected demographic groups
- Victim and perpetrator age patterns
- Gender-based crime analysis
- Relationship trends between victims and offenders
- Racial distribution analysis
